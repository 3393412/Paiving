🏠 PRIMARY SOURCE — Indoor IoT Sensor (1-minute resolution)
This sensor is installed indoors and is the primary data source (highest temporal resolution). Role in this system: (1) establish an indoor air quality baseline for the location, (2) detect rising PM2.5 trends before hourly outdoor data updates, (3) compute variability (CV%) to assess measurement confidence, (4) provide indoor context for interpretation of outdoor conditions.
Indoor PM2.5 is NOT used to estimate outdoor PM2.5 — real outdoor data comes from OpenAQ.

📋 Running Score Formula — Full Specification (Standard-Based)
Formula (exact code match)
// Base = 100 · Score = max(0, 100 − Σ penalties)

// ① AQI-US (IQAir · US EPA AQI · max −45)
AQI 0–50 → −0 // Good
AQI 51–100 → −10 // Moderate
AQI 101–150 → −20 // Unhealthy·Sensitive
AQI 151–200 → −35 // Unhealthy
AQI >200 → −45 // Very Unhealthy/Hazardous

// ② PM2.5 support (OpenAQ · EPA tiers · max −10)
PM ≤12.0 → −0 // Good
PM 12.1–35.4→ −3 // Moderate
PM 35.5–55.4→ −6 // Unhealthy·Sensitive
PM >55.4 → −10 // Unhealthy
// AQI+PM capped at 50 pts total

// ③ Temperature (ACSM tropical · max −20)
T <30°C → −0 // Ideal
T 30–33°C → −5 // Warm — hydrate
T 34–36°C → −10 // Hot — Bangkok avg
T 37–39°C → −15 // Very Hot
T ≥40°C → −20 // Extreme

// ④ Heat Stress (Steadman 1979 · max −8)
// Penalty ONLY when temp is also elevated
H≥85% && T≥30 → −8
H≥80% && T≥30 → −5
H≥75% && T≥28 → −3
else → −0 // no effect without heat

// ⑤ UV Index (WHO standard · max −15)
UV 0–2 → −0 // Low
UV 3–5 → −5 // Moderate
UV 6–7 → −10 // High
UV 8–10 → −13 // Very High
UV ≥11 → −15 // Extreme

// ⑥ Rain probability (OWM · toggleable · max −15)
Rain <20% → −0
Rain 20–49% → −5
Rain 50–69% → −10
Rain ≥70% → −15

Score = max(0, 100 − Σ penalties)
Standards & Justification
// AQI source standard
US EPA AQI Technical Assistance
Document (2024) · NowCast method
AQI ≠ PM2.5 — different metrics

// PM2.5 standard
US EPA breakpoints (µg/m³):
Good <12 · Moderate 12–35
USG 35–55 · Unhealthy >55
WHO guideline: 15 µg/m³ (24h)

// Temperature standard
ACSM Position Stand: Exertional
Heat Illness (2007)
Adapted for Bangkok (avg 30–35°C)
Penalty starts 30°C not 35°C

// Heat stress standard
Steadman (1979) Heat Index:
at 32°C + 80% RH →
apparent temp = 39°C (danger)
Source: ACSM Heat Illness 2007

// UV standard
WHO Global Solar UV Index:
A Practical Guide (2002)

// Score thresholds
80–100 → Excellent · Safe
60–79 → Good · Precautions
40–59 → Moderate · Shorten run
20–39 → Poor · Avoid strenuous
0–19 → Hazardous · Do not run
